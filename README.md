# LegalHub — Herramienta para Juristas y Litigantes del Derecho Paraguayo

Plataforma de consulta de códigos legales vigentes del Paraguay, organizada por categorías jurídicas.

## 🗂️ Categorías

1. **Códigos Civiles, Comerciales y de Familia**
   - Código Civil (Ley N° 1183/85)
   - Código de la Niñez y la Adolescencia (Ley N° 1680/01)

2. **Códigos Penales y de Ejecución**
   - Código Penal (Ley N° 1160/97)
   - Código de Ejecución Penal (Ley N° 5162/14)

3. **Códigos Laborales**
   - Código Laboral (Ley N° 213/93)

4. **Códigos de Procesos o Procedimientos**
   - Código Procesal Civil (Ley N° 1337/88)
   - Código Procesal Penal (Ley N° 1286/98)
   - Código Procesal Laboral (Ley N° 742/61)

5. **Códigos Institucionales y de Organización**
   - Código de Organización Judicial (Ley N° 879/81)
   - Código Electoral (Ley N° 834/96)

6. **Códigos de Navegación, Espacio y Transporte**
   - Código de Navegación Fluvial y Marítima (Ley N° 476/57)
   - Código Aeronáutico (Ley N° 1860/02)

7. **Códigos Sectoriales y de Recursos Naturales**
   - Código Rural (Ley N° 1248/31)
   - Código Aduanero (Ley N° 2422/04)
   - Código de Minería (Ley N° 3180/07)
   - Código Sanitario (Ley N° 836/80)
